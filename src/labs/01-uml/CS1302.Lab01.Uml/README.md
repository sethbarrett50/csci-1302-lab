# Lab 01 — UML Practice (Ungraded)

This is an **optional, ungraded** practice lab.

## Goal
Practice going from a UML class diagram to C# classes.

## What to do
1. Create a UML diagram for `Book` and `Library` (see `docs/practice/uml/practice-uml-01.md`)
2. Implement the classes in `Models/`
3. Use `Program.cs` to test behavior

## Run
From repo root:
```bash
make run LAB=src/labs/01-uml/CS1302.Lab01.Uml
```
