# Practice Labs (Ungraded)

These are **optional, ungraded practice labs** for CS 1302 students who want extra reps or need a slower pace.

## How to use this repo
1. Pick a topic below
2. Read the practice prompt
3. Implement in the matching `src/labs/...` project (we’ll set that up next)
4. Use the “Self-check” list to verify your work

## Topics
- [UML Diagrams](practice/uml/README.md)
- [Arrays (1D)](practice/arrays-1d/README.md)
- [Arrays (2D)](practice/arrays-2d/README.md)
- [Properties & Exceptions](practice/properties-exceptions/README.md)

## What “ungraded” means
- No submissions required unless your instructor asks
- These are here to help you practice and build confidence
- You can ask questions in office hours / lab time using your attempt as a starting point
