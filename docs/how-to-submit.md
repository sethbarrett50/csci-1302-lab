# How to “Submit” (Optional)

These practice labs are **ungraded**, so you typically do **not** need to submit anything.

If your instructor asks you to share your work:
- Push your changes to your GitHub repo (or a branch)
- Or zip the specific lab folder under `src/labs/...`

If you’re asking for help, be ready to share:
- What you expected to happen
- What actually happened (error message / output)
- The smallest code snippet that shows the issue
