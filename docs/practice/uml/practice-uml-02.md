# Practice UML 02 — “Student + Course”

## Goal
Model a simple relationship between students and courses.

## Requirements
Create UML for:
- `Student` (Id, Name)
- `Course` (Code, Title)
- `Enrollment` (Student, Course, Grade?)

Implement in C#:
- Create a few students/courses
- Create enrollments
- Print a student schedule
